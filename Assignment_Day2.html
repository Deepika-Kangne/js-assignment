Question-2 : Explain with examples the remaining Methods of Strings and Arrays?

[A].Strings Methods

1.split():A string can be converted to an array with the split() method

2.subString():substring() is similar to slice().The difference is that substring() cannot accept negative indexes.

3.replaceAll():The replaceAll() method returns a new string with all matches of a pattern replaced by a replacement.

4.replace():The replaceAll() method returns a new string replaces only the first match of a pattern replaced by a replacement.

5.trim():The trim() method removes whitespace from both sides of a string.

[B].Array Methods

1.toString() : converts an array to a string of (comma separated) array values.

2. join() : method also joins all array elements into a string.

3. pop() : method returns the value that was "popped out".

4.push() : method adds a new element to an array (at the end).

5.splice() : method can be used to add new items to an array.
